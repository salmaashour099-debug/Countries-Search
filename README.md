# 🌍 Countries Search

A responsive web application that allows users to search for countries and view detailed information such as the flag, capital city, population, and region using the REST Countries API.

## 🚀 Live Demo

[https://salmaashour099-debug.github.io/countries-search/](https://salmaashour099-debug.github.io/Countries-Search/)

---

## 📋 Features

* Search countries by name
* Display country flag
* View country capital city
* View population statistics
* View country region
* Loading indicator while fetching data
* Error handling for invalid country names
* Responsive Bootstrap UI

---

## ⚙️ How It Works

### Search

Users enter the name of a country and click the search button.

### Fetch Data

The application sends a request to the REST Countries API to retrieve country information.

### Display Results

The application dynamically displays:

* Country Flag
* Country Name
* Capital City
* Population
* Region

### Error Handling

If the country does not exist, an error message is displayed to the user.
